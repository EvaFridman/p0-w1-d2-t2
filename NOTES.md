name - Идентификатор элементов, необходимый для корректного определения полей сервером. Также используется для группировки радиокнопок, обращения к элементам из JS.

action - Атрибут, в котором указывается ссылка, по которой браузер должен отправить данные из формы.

method - Атрибут, определяющий HTTP-метод, который будет использоваться браузером для отправки данных из формы на сервер.

GET - Используется для запроса данных с сервера. При передаче данных информация передаётся в URL, информация видна в адресной строке и никак не скрыта/зашифрована.

POST - Используется для отправки данных на сервер. Информация передаётся в теле HTTP-запроса, а не в URL, скрыта.
